# Tiny Prompters Docs

Mintlify documentation workspace for Tiny Prompters projects, agent workflows, and product/venture operating notes.

## What It Contains

- Documentation navigation in `docs.json`
- Getting-started and development pages
- AI tool setup guides
- API reference scaffold
- Reusable Mintlify snippets and assets

## Run Locally

Install the Mintlify CLI:

```bash
npm i -g mint
```

Preview the docs:

```bash
mint dev
```

Open [http://localhost:3000](http://localhost:3000).

## Publishing

This repo is intended to publish through Mintlify's GitHub integration. Keep product-specific docs, API reference material, and AI-agent setup guidance in this repo rather than leaving it as a generic starter kit.
